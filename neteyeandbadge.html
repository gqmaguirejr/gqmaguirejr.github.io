Remote rebooting via the Axis Communications NetEye

Attachment to the camera (huge)
Attachment to the badge (huge)

Rebooting is done by accessing the URL

    http://cameraname/io/relayon.txt
followed by the URL
    http://cameraname/io/relayoff.txt

Note that you have to be logged in to the camera server as the root user to use these functions.
